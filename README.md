# Descripción 🌴
Proyecto donde utilizo **PHP**, **JavaScript**, **CSS** y **PostgreSQL**, junto a la arquitectura **MVC** (Model-View-Controller), para hacer una web sobre discos de Vinilo con manejo de datos de usuario y de productos.
